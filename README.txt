# Ai-N-Train Project

Welcome to the Ai-N-Train project! Ai-N-Train is an innovative visualization and onboarding platform designed to help you train, monitor, and manage AI models in an interactive and intuitive way. With a strong focus on visual workflows and modular components, Ai-N-Train allows users to interactively create and refine their AI goals, seeing them evolve in real-time.

## Key Features
- **Interactive AI Training Interface**: A seamless, user-friendly interface where you can input prompts and watch them transform into actionable AI workflows.
- **Visualization Dashboard**: Track your progress visually through a dynamic dashboard that adapts as new prompts are added or modified.
- **Modular Goals and Connections**: Generate central nodes that represent your training goals, with dynamic links that help visualize the steps and connections necessary to achieve them.
- **Customizable Training Parameters**: Flexibility in adjusting key AI training settings, making it easier to align the model training process with your specific needs.
- **Integration with Various AI Models**: Compatible with multiple AI models, ensuring a flexible foundation for various projects.
- **Evolution Through User Interactions**: Built on a RAG (Retrieval-Augmented Generation) database, the system learns and evolves based on user interactions, offering a sustainable and adaptive experience for users.

## Getting Started
To get started with the project, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/Ai-N-Train.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Ai-N-Train
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:3000` to access the application.
